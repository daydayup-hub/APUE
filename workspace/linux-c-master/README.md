# LinuxC

#### 介绍
代码来源于李慧芹老师讲解的基于LINUX的C语言课程，包含数据结构与算法，linux编程，俄罗斯方块，流媒体广播项目等代码。
代码测试环境：Ubuntu18.04
视频地址：https://www.bilibili.com/video/BV18p4y167Md 。

#### 软件架构
软件架构说明

base:             c 基础编程

data_structure:   数据结构与算法

file: 			  linux文件相关操作

io: 			   linux IO相关操作

ipc： 				linux IPC相关操作

minproj:			c项目基础组成

minproj_extern:		extern关键字使用案例

parallel:			并发编程（信号，线程使用）

process:			进程相关操作

proj:				俄罗斯方块，流媒体广播项目
